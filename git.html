git 환경설정

1. 깃 로그인 하고 레파지토리 만들기 (public으로 하면 공유하기 매우 좋다)
2. 깃 설치하기
3. git bash로 들어가서 이와 같은 명령어를 입력해줍니다.
(1) git config --global user.name "jiwoo choi" (내 영어이름)
(2) git config --global user.email "w2040@e-mirim.hs.kr" (깃 허브에 가입할 때 사용했던 이메일)
(3) git config --list (설정 확인)

commit 방법(첫 번째로 커밋했을 때)

1. terminal -> new terminal 을 눌러 terminal을 생성해줍니다.
2. terminal에 이와 같은 명령어들을 순서대로 입력해줍니다.
(1)git init (2)git add . (3) git status<선택> (4)git commit -m "first commit"<히스토리> (5)repository에 있는 remote부분을 복사해서 넣기 (6)git remote -v<확인용> (7)git push origin master
